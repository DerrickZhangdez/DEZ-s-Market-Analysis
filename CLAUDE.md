# DEZ Market Analysis - Claude Code Instructions

## Git Behavior
- ALWAYS commit directly to `main` — never create a new branch
- After generating the report, stage all new files, commit, and push to `main`
- Commit message format: "Daily market report: YYYY-MM-DD"
- Do NOT open pull requests

## Report Output
- Save daily reports to `/reports/YYYY-MM-DD.md`
- Save weekly previews to `/weekly/YYYY-MM-DD.md`
